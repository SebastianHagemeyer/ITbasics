Student Work Feedback

Upload a photo or scan of completed quiz work. The tool finds the question regions, reads each one, compares it to the answer key, and writes back feedback next to the cropped image of that question.

  1. Upload an image of the student's work.
  2. Pick a module so the tool knows which answer key to use.
  3. Auto-detect question regions, or drag on the image to draw your own boxes.
  4. Analyse to get AI-style feedback per question, with the cropped image shown alongside.
  5. Export the feedback as a printable report.

Drop in an image to get started. Once it loads, drag on it to draw a box around a question, or click Auto-detect regions.
